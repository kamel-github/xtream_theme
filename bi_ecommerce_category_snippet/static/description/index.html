E-Commerce Category Snippet Designer Odoo Apps

Product Category Snippet Builder odoo app provide features to display different e-commerce category with numbers of category snippets with different styles, user have options to display category image, category descriptions and product count.

Features

E-Commerce Category Snippets.

User can display stylish e-commerce category snippets on webpage.

Drag and Drop Features.

User just have to drag e-commerce category snippet and drop where he/she wants to display.

Different eCommerce Category Snippet Style.

User can display e-commerce category in different 23 styles.

Option to Display Category Image, Description and Count.

User have option to display category image, decriptions and product count.
